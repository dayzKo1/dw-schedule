# dw-schedule · 周行程实时备份

度威客户拜访/销售行程的可视化周表，自动备份。

## 部署

- **GitHub Pages**: https://dayzKo1.github.io/dw-schedule/
- **源 HTML**: `index.html`（单文件，自包含）
- **更新流程**: 改 `index.html` → `git add .` → `git commit` → `git push` → Pages 自动部署

## 客户端点访问

无需登录，直接打开 URL 即可。所有行程数据都内嵌在 HTML 里。

## 内容性质

- 真实客户公司名（宁德洲际 / 福州赛威 / 福建普创 / 香橼橙黄 / 众前友美）
- 销售/拜访记录
- 时间安排

仓库公开（Pages 免费版要求），外人能搜到，但没人知道 URL 就找不到。
